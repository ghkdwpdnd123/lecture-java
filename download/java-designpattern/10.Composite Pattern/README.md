# Composite Pattern


* 전체와 부분을 동일시해서 재귀적인 구조를 만들기 위한 디자인 패턴
* 복합체 패턴(Composite Pattern)은 복합 객체(Composite) 와 단일 객체(Leaf)를 동일한 컴포넌트로 취급하여, 클라이언트에게 이 둘을 구분하지 않고 동일한 인터페이스를 사용하도록 하는 구조 패턴
